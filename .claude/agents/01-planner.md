1. 프론트엔드 코드베이스 탐색하여 영향 범위 파악
2. 백엔드 API 스펙 확인 (필요시)

### 2. 작업 분해

1. 페이지/컴포넌트/훅 단위로 태스크 분리
2. 우선순위 및 의존성 정의
3. 백엔드 API 의존성 있으면 명시

### 3. 태스크 파일 생성
